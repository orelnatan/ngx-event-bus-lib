# For the lib - NgxEventBus

## Building

To build the library, run:

```bash
ng build ngx-event-bus
```

Or for Production:

ng build ngx-event-bus-lib --configuration production








## Running unit tests

To execute unit tests with the [Karma](https://karma-runner.github.io) test runner, use the following command:

```bash
ng test
```

## Running unit-tests(lib)

Run the command: ng test ngx-event-bus-lib

## Running unit-tests with coverage(lib)

Run the command ng test ngx-event-bus-lib --coverage
